# Rubyish Omega2 GPIOs

A wrapper to control [Onion Omega2](https://onion.io/omega2/) GPIOs in ruby

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'omega2_gpio'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install omega2_gpio
